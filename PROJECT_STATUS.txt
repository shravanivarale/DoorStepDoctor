                            🚀 DEPLOYMENT STEPS
================================================================================

STEP 1: PASTE AWS KEYS
  📝 Edit: backend/.env
  🔑 Paste: BEDROCK_KB_ID
  🔑 Paste: BEDROCK_GUARDRAIL_ID
  🔑 Paste: COGNITO_USER_POOL_ID
  🔑 Paste: COGNITO_CLIENT_ID

STEP 2: INSTALL & BUILD BACKEND
  $ cd backend
  $ npm install
  $ npm run build

STEP 3: DEPLOY TO AWS
  $ npm run deploy:dev
  💾 Save API Endpoint URL!

STEP 4: CONFIGURE FRONTEND
  📝 Edit: .env (root directory)
  🔗 Paste: REACT_APP_API_ENDPOINT
  🔑 Paste: REACT_APP_COGNITO_USER_POOL_ID
  🔑 Paste: REACT_APP_COGNITO_CLIENT_ID

STEP 5: START FRONTEND
  $ cd ..
  $ npm install
  $ npm start
